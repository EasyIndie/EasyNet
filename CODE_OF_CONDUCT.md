# 贡献者公约行为准则

## 我们的承诺

作为成员、贡献者和社区领导者，我们承诺让每个人的参与体验
免受骚扰，无论年龄、体型、残障、族裔、性别特征、性别认同与表达、
经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教或性取向。

我们承诺以促进开放、热情、多元、包容、健康社区的方式行事和互动。

## 我们的标准

有助于营造积极社区环境的行为示例：

- 对他人展现同理心和善意
- 尊重不同意见、观点和经历
- 给予并优雅接受建设性反馈
- 为自己的错误承担责任，向受影响者道歉，并从中吸取教训
- 关注对整体社区最好的结果，而非仅对个人有利

不可接受的行为示例：

- 使用性化语言或图像，以及任何形式的性关注或越界行为
- 挑衅、侮辱或贬损性评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人隐私信息，如地址或邮箱
- 在职业环境中可合理认定为不当的其他行为

## 执行责任

社区领导者负责明确和执行我们的行为标准，对于任何被认为不当、
具有威胁性、冒犯性或有害的行为，将采取适当和公平的纠正措施。

## 适用范围

本行为准则适用于所有社区空间，当个人在公共场合正式代表社区时
同样适用。

## 执行

如有辱骂、骚扰或其他不可接受的行为，可向负责执行的社区领导者
报告。所有投诉将得到及时和公正的审查和调查。

## 执行指南

社区领导者将遵循以下社区影响指南，决定对违反本行为准则的行为
采取何种处罚：

### 1. 纠正

**社区影响：** 使用不当语言或存在其他在社区中被视为不专业
或不受欢迎的行为。

**处罚：** 社区领导者发出私密书面警告，明确说明违规性质
并解释为何该行为不当。可能要求公开道歉。

### 2. 警告

**社区影响：** 通过单次事件或一系列行为造成的违规。

**处罚：** 警告并附继续行为的后果。在指定时间段内禁止与涉事
人员互动，包括主动与执行行为准则的人员互动。这包括在社区
空间以及社交媒体等外部渠道中避免互动。违反这些条款可能导致
临时或永久封禁。

### 3. 临时封禁

**社区影响：** 严重违反社区标准，包括持续不当行为。

**处罚：** 在指定时间段内禁止与社区进行任何形式的互动或公开
交流。此期间不允许与涉事人员进行公开或私下互动，包括主动
与执行行为准则的人员互动。违反这些条款可能导致永久封禁。

### 4. 永久封禁

**社区影响：** 表现出违反社区标准的模式，包括持续不当行为、
骚扰个人、或攻击或贬低特定群体。

**处罚：** 永久禁止在社区内进行任何形式的公开互动。
